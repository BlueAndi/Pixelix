ShellyPlugSPlugin

The plugin shows the current AC power being drawn via a Shelly PlugS, in watts.

REST API

Get ip-address

GET {{ORIGIN}}/rest/api/v1/display/uid/<PLUGIN-UID>/ipAddress

Set ip-address

POST {{ORIGIN}}/rest/api/v1/display/uid/<PLUGIN-UID>/ipAddress?set=<IPADDRESS>
  • PLUGIN-UID: The plugin unique id.
  • IPADDRESS: The ip-address for the Shelly PlugS server.
