WeatherPlugin

The plugin shows the weather forecast for a configured location.

REST API

Get location

GET {{ORIGIN}}/rest/api/v1/display/uid/<PLUGIN-UID>/weather
  • PLUGIN-UID: The plugin unique id.

Set location

POST {{ORIGIN}}/rest/api/v1/display/uid/<PLUGIN-UID>/weather?loc=<LOCATION>&days=<DAYS>&APIKEY=<APIKEY>
  • PLUGIN-UID: The plugin unique id.
  • LOCATION: The location.
  • DAYS: The latitude of the location.
  • APIKEY: The APIKey.
