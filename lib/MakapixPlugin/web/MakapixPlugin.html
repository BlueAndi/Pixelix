MakapixPlugin

This plugin is a Makapix player (https://makapix.club) that plays pixel artwork.

Playback Control

Makapix

The player key for the Makapix player.
The MQTT instance number to use. 0 is the primary instance, see MQTT service configuration.

To use the Makapix plugin, you need to create a free account at Makapix Club and get a player key from there. The player key is used to identify your display device.

After you have created an account and obtained a player key, you can configure the plugin by entering the player key in the configuration form below.

Additionally, you need to specify which MQTT instance the plugin should use to communicate with the Makapix service. Make sure that the MQTT service is properly configured and running.

Broker hostname: makapix.club
Broker port: 8883 (secure connection required)
User: Your player key
Password: (leave empty)
Root CA Certificate: Download it from your Makapix Club account.
Client Certificate: Download it from your Makapix Club account.
Client Key: Download it from your Makapix Club account.

The Makapix plugin requires registration with the Makapix service to function properly. This process involves provisioning the Pixelix device with a unique player key.
