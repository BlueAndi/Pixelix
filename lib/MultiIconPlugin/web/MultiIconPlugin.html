MultiIconPlugin

The plugin is able to show multiple different icons.

Each icon can be set separately via the REST API.

REST API

Get icon file id of slot

GET {{ORIGIN}}/rest/api/v1/display/uid/<PLUGIN-UID>/slot/<SLOT-ID>
GET {{ORIGIN}}/rest/api/v1/display/alias/<PLUGIN-ALIAS>/slot/<SLOT-ID>
  • PLUGIN-UID: The plugin unique id.
  • PLUGIN-ALIAS: The plugin alias name.
  • SLOT-ID: The id of the slot (starting with 0 from left ([0; 2])).

Set a icon file id of slot

POST {{ORIGIN}}/rest/api/v1/display/uid/<PLUGIN-UID>/slot/<SLOT-ID>?fileId=<FILE-ID>
POST {{ORIGIN}}/rest/api/v1/display/alias/<PLUGIN-ALIAS>/slot/<SLOT-ID>?fileId=<FILE-ID>
  • PLUGIN-UID: The plugin unique id.
  • PLUGIN-ALIAS: The plugin alias name.
  • SLOT-ID: The id of the slot (starting with 0 from left ([0; 2])).
  • FILE-ID: The id of the icon file.

Get all icons

GET {{ORIGIN}}/rest/api/v1/display/uid/<PLUGIN-UID>/slots
GET {{ORIGIN}}/rest/api/v1/display/alias/<PLUGIN-ALIAS>/slots
  • PLUGIN-UID: The plugin unique id.
  • PLUGIN-ALIAS: The plugin alias name.

Set all icons

POST {{ORIGIN}}/rest/api/v1/display/uid/<PLUGIN-UID>/slots?slots._0_.fileId=<FILE-ID>slots._1_.fileId=<FILE-ID>slots._2_.fileId=<FILE-ID>
POST {{ORIGIN}}/rest/api/v1/display/alias/<PLUGIN-ALIAS>/slots?slots._0_.fileId=<FILE-ID>slots._1_.fileId=<FILE-ID>slots._2_.fileId=<FILE-ID>
  • PLUGIN-UID: The plugin unique id.
  • PLUGIN-ALIAS: The plugin alias name.
  • SLOT-ID: The id of the slot (starting with 0 from left ([0; 2])).
  • FILE-ID: The id of the icon file.
