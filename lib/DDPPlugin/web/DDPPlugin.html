DDPPlugin

The plugin setup a server supporting the Distributed Display Protocol (DDP), which is used e.g. by xlights or LedFx.

Supported formats:

  • RGB with 24-bit per pixel

Add Ethernet controller

  • IP Address:<IP-ADDRESS>
  • Protocol: DDP
